Announcement for the Calendar App

Created by: Long Tran

April 12, 2020

My group member(Andy Phan) and I intend to create a full stack Calendar application. This idea arose as a result of our poor management of multiple work commitments. The Calendar application would assist us and others in staying on track. The app will allow the user to create an account in order to access their personal calendar. The user can then schedule an event for a specific time.
